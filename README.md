# Lok Drishti

## Description
Lok Drishti is a simple web application that helps citizens report and track public issues like broken roads, garbage problems, and street light issues.

## Features
- Users can report public issues
- Users can track the status of their complaints
- Authorities can view issues on a dashboard

## Technologies Used
- HTML
- CSS
- JavaScript

## Installation
1. Download or clone the project
2. Open the project folder
3. Run the `index.html` file in a browser

## Usage
- Open the website
- Submit a public issue using the form
- Track the status of the issue
- Authorities can manage issues from the dashboard
